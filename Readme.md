# Tennis
It consists mainly of a class that contains the methods for a yatzy game.\
It was programmed using the TDD method.

## Build with
* .NET Core v3.0.0
* #C
* XUnit v2.4.1
* A lot of fun

## Installation
* Install latest [.Net SDK](https://dotnet.microsoft.com/download)

### Install dependencies
```shell
dotnet restore
```

## Run unit tests
```shell
cd .\Yatzy.Test
dotnet test
```
